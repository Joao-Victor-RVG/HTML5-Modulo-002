Compatibilidade com navegadores e dispositivos

É um problema recorrente no mundo de desenvolvimento web, não só relacionado exclusivamente a tipográfia, por isso é importante usar algo que os proprios editores de código recomendam:

  • font-family: Arial, Helvetica, sans-serif

Como podemos ver o editor recomendam os 3 tipos de tipográfia, isso acontece pois alguns navegadores por padrão não interpretam alguns tipos, e com isso o navegador ou dispositivo pode ter pelo menos 3 oportunidades de tentativa de fazer essa interpretação.
